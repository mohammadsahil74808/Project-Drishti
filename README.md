# SentinelX AI
### AI-Driven Crime Analytics & Visualization Platform — Karnataka State Police Datathon 2026

SentinelX AI is a production-ready, AI-native crime intelligence platform combining
geospatial analytics, predictive forecasting, NLP-driven investigation tools, and
role-based command dashboards for Karnataka State Police.

## Stack (100% free/open-source)
- **Frontend:** React + Vite + TypeScript + Tailwind CSS
- **Backend:** FastAPI (Python)
- **Database:** PostgreSQL + PostGIS
- **Cache/Queue:** Redis
- **Containerization:** Docker + Docker Compose
- **CI/CD:** GitHub Actions
- **IDE:** VS Code

## Repository Layout
| Folder | Purpose |
|---|---|
| `backend/` | FastAPI application — API, AI/ML pipelines, database layer |
| `frontend/` | React + Vite dashboard application |
| `data/` | Synthetic data generators and seed datasets |
| `ml_notebooks/` | Exploratory data analysis and model experimentation |
| `docs/` | Setup guides, standards, architecture documentation |
| `scripts/` | Developer utility scripts |
| `.github/` | CI workflows and issue/PR templates |
